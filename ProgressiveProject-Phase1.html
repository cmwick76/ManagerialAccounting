üìç Tutorial 1: Cell Referencing

Cell referencing is the foundation of Excel. Instead of typing numbers, you link to cells containing the data. This makes your spreadsheet dynamic and easy to update.

Why Use Cell References?

If you type the number 125000 into a formula and later the value changes, you have to manually update every formula. With cell references, you change the data once and all formulas update automatically!

Referencing Cells on the Same Tab

  1. Click the cell where you want the formula
  2. Type an equals sign = to start the formula
  3. Click the cell you want to reference (or type the cell address like C5)
  4. Press Enter

Example: Calculating Raw Materials Used

Cell B5 contains Beginning Inventory: $125,000

Cell B6 contains Purchases: $456,000

In cell B7, enter: =B5+B6

Result: $581,000

Referencing Cells on Different Tabs

To reference data from another worksheet, use this format: ='Sheet Name'!CellAddress

  1. Type an equals sign =
  2. Type the sheet name in single quotes: 'Company Data'
  3. Type an exclamation mark !
  4. Type or click the cell address C5
  5. Complete formula: ='Company Data'!C5
// Linking to data from another tab ='Company Data'!C5 ‚Üí Gets value from Company Data tab, cell C5 ='COGM'!B18 ‚Üí Gets COGM total from COGM tab, cell B18 ='Income Statement'!B21 ‚Üí Gets Net Income from Income Statement tab
üí° PRO TIP: Click the tab name instead of typing it! Excel will automatically format the reference correctly. Type =, click the other tab, click the cell, press Enter. Done!
‚ö†Ô∏è COMMON MISTAKE: Forgetting the single quotes around sheet names with spaces. =Company Data!C5 will give an error. You need: ='Company Data'!C5

üîí Tutorial 2: Absolute vs. Relative References

Understanding the difference between absolute and relative references is crucial when copying formulas.

Relative References (Default)

When you copy a formula, Excel automatically adjusts the cell references. For example, if =B5 is in cell C5 and you copy it down to C6, it becomes =B6.

Absolute References (Lock with $)

Use the $ symbol to lock a reference so it doesn't change when copied. Press F4 while editing a cell reference to toggle between reference types.

Reference Type Example What's Locked When to Use
Relative =B5 Nothing Copying formulas that should adjust
Absolute =$B$5 Both column & row Always reference the same cell
Mixed (column) =$B5 Column only Lock column, let row change
Mixed (row) =B$5 Row only Lock row, let column change

Step-by-Step: Using F4 to Add $ Symbols

  1. Click in the cell containing your formula
  2. Click in the formula bar and position cursor on the cell reference (like B5)
  3. Press F4 once ‚Üí $B$5 (absolute)
  4. Press F4 again ‚Üí B$5 (mixed - row locked)
  5. Press F4 again ‚Üí $B5 (mixed - column locked)
  6. Press F4 again ‚Üí B5 (back to relative)

Example: When linking to the Company Data tab, you typically want absolute references:

='Company Data'!C5 ‚Üí might shift when copied

='Company Data'!$C$5 ‚Üí stays locked on C5 ‚úì

üí° BEST PRACTICE: For this project, use absolute references when linking to the Company Data tab. This prevents errors if you copy formulas around.

üßÆ Tutorial 3: Creating Formulas

Excel formulas perform calculations on your data. All formulas start with an equals sign =

Basic Arithmetic Operations

Operation Symbol Example Result
Addition + =B5+B6 Sum of B5 and B6
Subtraction - =B10-B11 B10 minus B11
Multiplication * =B5*0.25 B5 times 0.25
Division / =B15/B16 B15 divided by B16

The SUM Function

Instead of =B5+B6+B7+B8, use =SUM(B5:B8) to add a range of cells.

  1. Type =SUM(
  2. Click and drag to select the range of cells, or type B5:B8
  3. Type the closing parenthesis )
  4. Press Enter
// SUM examples for this project =SUM(B5:B7) ‚Üí Adds cells B5, B6, and B7 =SUM(B5,B6,B8) ‚Üí Adds specific cells (not consecutive) =SUM(B5:B7,B10:B12) ‚Üí Adds multiple ranges

Real Example from COGM:

Total Manufacturing Costs = RM Used + Direct Labor + Factory Overhead

B9 = Raw Materials Used ($483,000)

B11 = Direct Labor ($389,000)

B12 = Factory Overhead ($267,000)

Formula in B13: =SUM(B9,B11,B12)

Result: $1,139,000

The AVERAGE Function

Calculates the average (mean) of a range of numbers.

// AVERAGE examples - useful for inventory analysis =AVERAGE(B5,B6) ‚Üí Average of two cells =AVERAGE(B5:B10) ‚Üí Average of a range

Real Example from Inventory Analysis:

Average RM Inventory = (Beginning RM Inventory + Ending RM Inventory) / 2

B7 = Beginning RM Inventory ($125,000)

B8 = Ending RM Inventory ($98,000)

Formula in B9: =AVERAGE(B7,B8)

Result: $111,500

üí° ORDER OF OPERATIONS: Excel follows PEMDAS (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction). Use parentheses to control the order: =(B5+B6)/2 vs =B5+B6/2 give different results!

üí∞ Tutorial 4: Professional Number Formatting

Proper number formatting makes your spreadsheets professional and easy to read.

Method 1: Quick Format Buttons (Ribbon)

  1. Select the cells you want to format
  2. Go to the Home tab in the ribbon
  3. Look for the Number group
  4. Click the appropriate button:
    • $ (Currency): Adds dollar sign and 2 decimals
    • % (Percent): Converts to percentage
    • , (Comma): Adds thousand separators
    • Increase/Decrease Decimal: Adjusts decimal places

Method 2: Format Cells Dialog (More Control)

  1. Select the cells to format
  2. Right-click and choose Format Cells (or press Ctrl+1)
  3. Click the Number tab
  4. Choose a category:
    • Currency: For dollar amounts with $ symbol
    • Number: For ratios and general numbers
    • Percentage: For percentages
  5. Set decimal places (usually 0 for currency, 2 for ratios)
  6. Click OK

Formatting Guide for This Project

Data Type Format Example Display How to Apply
Dollar Amounts Currency, 0 decimals $1,139,000 Select cells ‚Üí Click $ button
Inventory Turnover Number, 2 decimals 4.33 Format Cells ‚Üí Number ‚Üí 2 decimals
Days' Sales Number, 1 decimal 84.3 Format Cells ‚Üí Number ‚Üí 1 decimal
Percentages (if used) Percentage, 1 decimal 38.6% Click % button ‚Üí Adjust decimals
üí° KEYBOARD SHORTCUT: Press Ctrl+1 to open Format Cells dialog instantly! This works for any selected cell or range.
‚ö†Ô∏è COMMON MISTAKE: Don't type the $ symbol or commas when entering currency values in formulas! The $ is used for absolute cell references (like $B$5), not for currency. Enter the number 125000 and let Excel format it as $125,000. If you type $125,000 as a value, Excel treats it as text, not a number.

Custom Number Format (Advanced)

For more control, use custom formats. In Format Cells ‚Üí Number ‚Üí Custom:

$#,##0 ‚Üí Currency with no decimals: $1,139,000 $#,##0.00 ‚Üí Currency with 2 decimals: $1,139,000.00 0.00 ‚Üí Number with 2 decimals: 4.33 0.0 ‚Üí Number with 1 decimal: 84.3

üé® Tutorial 5: Professional Table Formatting

Professional formatting makes your work easy to read and visually appealing.

Adding Borders

  1. Select your table range (include headers and data)
  2. Go to Home tab ‚Üí Find the Borders button (grid icon)
  3. Click the dropdown arrow next to Borders
  4. Choose "All Borders" to add borders around every cell
  5. For thicker outer border: Select range ‚Üí Choose "Thick Box Border"

Applying Color to Headers

  1. Select your header row (row with column titles)
  2. Go to Home tab ‚Üí Find Fill Color (paint bucket icon)
  3. Choose Blue (use #0033A0 for Apex brand blue)
  4. Select header row again ‚Üí Find Font Color
  5. Choose White for text color
  6. Make Bold: Click the Bold button or press Ctrl+B

Alignment Matters

Content Type Alignment Why How to Apply
Text/Labels Left Easier to scan/read Home ‚Üí Align Left
Numbers/Currency Right Lines up decimal points Home ‚Üí Align Right
Headers Center Balanced appearance Home ‚Üí Center
Titles Left or Center Personal preference Home ‚Üí Center or Left

Adjusting Column Widths

  1. Hover between column letters (e.g., between B and C)
  2. Cursor changes to ‚Üî (double-headed arrow)
  3. Double-click to auto-fit to content
  4. OR drag to manually set width
  5. OR select columns ‚Üí Right-click ‚Üí Column Width ‚Üí Enter number (like 20)

Professional Formatting Checklist

  • ‚úì Headers: Blue background (#0033A0), white text, bold, centered
  • ‚úì All tables have borders
  • ‚úì Text is left-aligned, numbers are right-aligned
  • ‚úì Currency formatted with $ and commas

üè∑Ô∏è Tutorial 6: Using Named Ranges (Optional Advanced Skill)

Named ranges let you give a cell or range a meaningful name, making formulas easier to read and maintain.

What is a Named Range?

Instead of =B5+B6-B8, you could use =BeginningRM+Purchases-EndingRM. Much clearer!

Creating a Named Range

  1. Select the cell or range you want to name
  2. Click in the Name Box (upper left, shows cell address like "C5")
  3. Type a name (no spaces, like BeginningRM or DirectLabor)
  4. Press Enter
  5. Use the name in formulas instead of the cell address

Alternative Method: Define Name Dialog

  1. Select the cell
  2. Go to Formulas tab ‚Üí Click "Define Name"
  3. Enter the name and click OK

Rules for Named Ranges

Rule Example
Must start with letter or underscore ‚úì Sales or _Total
‚úó 1Sales
No spaces allowed ‚úì DirectLabor or Direct_Labor
‚úó Direct Labor
Cannot be same as cell address ‚úó A1 or B52
Case doesn't matter Sales = SALES = sales

Using Named Ranges in Formulas

Without Named Ranges:

='Company Data'!$C$5+'Company Data'!$C$9-'Company Data'!$C$6

With Named Ranges:

Much easier to understand! ‚úì
